# Wanderlust---Airbnb-like-website


Wanderlust is a full-stack Airbnb-like rental listing web application built with Node.js, Express, MongoDB, EJS, and Bootstrap. Users can browse property listings, view listing details, upload listing images, create new listings, edit/delete their own listings, and leave reviews with ratings.

## Features

- User authentication and authorization
- Create, edit, and delete listings
- Upload listing images locally
- Add reviews and star ratings
- Category filters for listings
- Search listings by title, location, or country
- Interactive location map using Leaflet and OpenStreetMap
- Flash messages for user feedback
- Responsive UI with Bootstrap

## Tech Stack

- Node.js
- Express.js
- MongoDB
- Mongoose
- EJS
- Bootstrap
- Passport.js
- Multer
- Leaflet.js
